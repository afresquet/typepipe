# tmi.js Example

## Usage

From the root of TypePipe run:

```shell
npm install

npm run build

cd examples/tmi.js

npm install
```

Change the `.env.example` file to `.env`, place your [authentication credentials](https://twitchapps.com/tmi/), and run:

```shell
npm run dev
```

## Commands

- !ping
- !counter \<counter name>
